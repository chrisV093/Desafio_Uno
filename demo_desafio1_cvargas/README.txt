Desarrollado con Java8 + Spring boot + maven

Pre-requisitos: java + maven, puerto 9090 disponible y sin servicio asignado, 
				ejecutar GDD en la maquina local en el puerto  8080.

compilar con comando *mvn package*
ejecutar el jar que se encuentra en el directorio /target, el nombre es: demo_desafio1-0.0.1.jar
*java -jar .\demo_desafio1-0.0.1.jar*

refierase a la siguiente url para el consumo del servicio

http://localhost:9090/demo/solved
